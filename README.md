This is a user management React application .  
It includes functionality for login, pagination, search,alert,modal, edit, and delete operations with Bootstrap5,React,Axios,React-icons,google-font.

## 🚀 Features
- Login 
- paginate users
- Search users 
- Edit user 
- Delete users
- Modal 
- Alerts 

---

## 🛠️ Tech Stack

- React.js
- Axios
- Bootstrap 5
- React Icons
- Google Fonts
